The Problem

AI4EO datasets have no standard packaging

Earth Observation datasets for machine learning need data and metadata together. Users filter by cloud cover, region, time, split, solar zenith, and dozens of other variables before loading a single pixel. No existing format packages data, metadata, and structure as a queryable portable unit.

STAC stores one JSON per sample — a million samples, a million files. STAC-GeoParquet fixes queries but flattens the hierarchy and doesn't bundle data. TACO solves both.
